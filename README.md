# Programavimo kalbu teorijos projektai

### Apie repozitorija

Cia kaupiami 2015 PKT modulio studentu projektiniai darbai.

### Projekto nuostatos

Kiekviena projekto komanda sudaro 3 studentai.

Kiekviena komanda sukuria kataloga ir ten kaupia savo projekto failus. Kompiliuotu failu kelti nereikia, tik išeities tekstus ir projekto failus, taip, kad bet kada galetu parsisiusti ir sukompiliuoti projekta. Katalogo pavadinimas - komandos pavadinimas.
